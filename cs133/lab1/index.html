Commands:

  • cp: Copy a file. Requires two arguments, source and destination. An example might be
    cp ~nick/cs133/infotutorial ./infotutorial
  • chmod: changes access permissions. An example might be
    chmod 711 ./index.html
  • cd: Changes the working directory
    cd ..
  • ssh: to remotely connect to a linux machine.
    ssh u1632823@joshua.dcs.warwick.ac.uk
  • man: this command let's you access a manual page for a specific command. An example would be:
    man ls
  • ls: lists all the files in a directory.
    ls
  • touch: creates a plain file.
    touch index.html
  • lpr: submits a printing job.
    lpr -Pdcs006a info.pdf
