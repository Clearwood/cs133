    2. Citing

  2. Citing of a BibTeX reference is very easy. The command \cite{label} has to be used to cite but apart from that the only other thing to do is to set the bibliography and the referencing style at the end of the document.

    	\documentclass{IEEEtran}

    4. Algorithms

  4. Writing algorithms in LaTeX is pretty easy too, thanks to the easy to use algorithm environment. To use the environment, the algorithmic package has to be imported: \usepackage{algorithmic} Every algorithm in LaTeX is written within that algorithmic environment: 
    \begin{algorithmic}
\STATE $S=0$
\end{algorithmic}
    Wrapping this environment within an algorithm environment allows the user to give the algorithm a label and to freely place it on the page.

    5. Source Code listings

  5. Using the listings package, it is possible to directly include source code into a LaTeX document and to correctly color encode it. This environment is highly configurable which means that for example specific color settings can be made.

Commands:

  • \bibliography{file name}: defines a bibliography used for the document.
  • \bibliographystyle{style}: defines the referencing style.
    • algorithmic environment
  • \STATE text: a normal statement.
  • \IF{condition} \STATE{text} \ELSE \STATE{text} \ENDIF: the normal if statement.
  • \FOR{condition} \STATE{text} \ENDFOR: To display a basic for loop.
  • \WHILE{condition} \STATE{text} \ENDWHILE: a basic while loop.
  • \REPEAT \STATE{text} \UNTIL{condition}: a basic while loop.
  • \PRINT text: creates a print statement.
    • listings environment
  • \lstset{language=programming language}: sets the language in which the following code is written.
  • \lstinputlisting{source_filename}: allows to import code from a source file.
