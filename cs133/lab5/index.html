    2. Redirecting and using pipes

  2. Redirecting an output stream to a text file or using a text file as input was something I did before coming to university so I did not feel uncomfortable doing that. Redirecting output is fairly simple, a command is called using the greater than symbol to redirect the output. An example for the redirection of the output to would be:
    ls > list.txt
    An example to redirect input into a command would be:
    wc -l < list.txt
    Using pipes had not been anything new for me too. Basically it allows to give the standard output of one command to another command and than display the output of the last command.
    ls -lt | head
